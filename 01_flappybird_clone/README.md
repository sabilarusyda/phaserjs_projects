# Flappy Bird Clone

We will learn about basic game mechanics like:
- Physical forces, like a gravity, how they works
- Game objects, what forces you can apply to the game objects like for example valocity
- Debug the code

## Menu Scene

- Play button
- Score button
- Exit button

## Play Scene

- Score countdown
- Best score countdown
- Pause button

Game Mechanic:
- Player can go up and down on the screen via:
  - Clicking the screen
  - Clicking the space bar on the keyboard
- Gradually, as player going through the pipes, it will get harder and harder

## Pause Scene

- Continue button
- Exit button

## Continue Scene

- It will continue the paused game
- There will be 3,2,1 countdown before the game started again so player can prepare

# Score Scene

- Show the latest Best Score
- Return button

# Exit Scene

- It will ending the current game session, setup up the score and return the player to the Menu Scene
